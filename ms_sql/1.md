### Należy to zapisać do pliku .sql a następnie dodać do harmonogramu zadań

```sql
BACKUP DATABASE [skip] TO  DISK = N'<path>' WITH NOFORMAT, NOINIT,  NAME = N'skip-Full Database Backup', SKIP, NOREWIND, NOUNLOAD,  STATS = 10
GO
```
